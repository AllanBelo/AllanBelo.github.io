What’s Git?

Is a distributed control version system, multiplataform, free an open source made by Linus Torvalds, Ok? Whats is control version 😖? Is source code change management.

What’s GitHub?

Free platform (has paid plans) in it you can store your GIT repositorys in cloud ☁️

How to install and setup Git?

Let’s go!

  • Download e install the Git

    • Windows x86
    • Windows x64
    • Linux or Mac

  • Create a new folder in your user directori (Rename to “Your Project Name”);

  • Open Git Bash (Terminal in case MAC or Linux) in created folder;

  • Type it to create a Git Repository: 

    git init

  • Create a new GitHub account;

  • Crate a SSH key to conect your Git repository in GitHub, type it: 

    ssh-keygen -t rsa -b 4096 -C "your GitHub mail account"

  • After execute a last command, type Enter again to sava the key in “/Users/YourUser/.ssh/id_rsa” (Linux: “/home/YourUser/.ssh/id_rsa”);

  • Type a secure passwrod and type Enter again;

  • Open the archive “/Users/YourUser/.ssh/id_rsa.pub” and copy your conteint;

  • Open GitHub > Settings > SSH > New SSH key > Type a title > paste the content > Add SSH Key, Key to conect Git and Github done!;

Let’s create a GitHb repository:

  • Click in ‘+’ icon (top rigth corner ) > Change ‘New Repository’ > Type a name to repository > Change Public (Visible for world) or Private (Visible for you) > Click in ‘Create Repository’;
  • In Git Bash (Terminal) type: 
    git remote add origin https://github.com/YourUser/YourProjectName.git
    Now your local repository is conected with the GitHub repository 😄.

How to use?

Now it’s time for fun 😆:

git init
Use to: Create repository.

git clone url or directory
Use to: Copy a exist local or remote repository to actual repository. Note: git clone depends git init.

git add archhive.extencion
Use to: After alter archives or add new archives in folder (repository) use git add to seng alteration to staging (staging it’s similar a data buffer concepts).

git remote add archhive.extencion url
Use to: Add archive to remote repository staging.

git commit -m "Coment"
Use to: The commit comand generate complet version of local project, the versions organized are a time line, it’ possible navigate in commits history to restory versions. Note: It’s important use good coments in commits to easy identify aterations.

git pull origin master
Use to: If a local version is obsolet, use it’s command to get remote repository(GitHub) content.

git push
Use to: Send commit to remote repository.

git log
Use to: Show time line commit’s history (Commit code identifier, author, create date and coment’s commit).

git status
Use to: Show state of repository and staging.
